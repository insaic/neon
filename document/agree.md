### agree
勾选按钮。

---

#### 例子
```html
<sq-agree v-model="value">我已阅读并接受《投保须知》</sq-agree>
```
---

#### API
| 参数           | 类型        | 说明                               | 默认值  |
|:-------------- |:---------- |:---------------------------------- |:------- |
| value          | `Boolean`   | 绑定值                             |         |
